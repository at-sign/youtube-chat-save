# youtube-chat-save
A js bookmarklet you can use to save a copy of the currently loaded live-chat log on a youtube page.

# What it does
The result will be that it presents two options in a window alert, and you choose whether to copy the chat contents to your clipboard or save to an html file. That's it!

# How to use
If this is your first time with a javascript bookmarklet, its use is simple.
It is activated by clicking a link you've added to your browser bookmarks, which triggers the code to run on the page you are on.

Open the txt file to view it, then copy the entire code block to your clipboard. Then go to your browser, and make a new bookmark, and enter the code block as the destination url for the bookmark. Save the bookmark, then go to the page you want to use it on, and click the bookmark.
